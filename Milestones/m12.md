- Gemini

# VECTOR DATABASE

Use PostgreSQL with pgvector.

Requirements:

- Vector storage
- Similarity search support
- Index optimization
- Metadata storage

Design for millions of vectors.

- GitHub Issues

- Commits
- Notion Pages
- Google Docs
- PDFs
- Markdown
- Text Files

Each chunk should include:

- Chunk ID
- Knowledge Item ID
- Content
- Metadata
- Source
- Token Count
