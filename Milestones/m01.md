# PROJECT CONTEXT

KnowWhy is an AI-powered Organizational Intelligence Platform.

KnowWhy connects software engineering tools such as:

- GitHub
- Notion
- Google Drive
- Google Calendar

KnowWhy builds an Organizational Memory using Retrieval-Augmented Generation (RAG) and semantic search.

The final product will allow users to ask questions such as:

> Why was PostgreSQL selected?

> Who implemented Authentication?

> Which meeting approved this architecture?

The AI should answer using evidence gathered from connected systems.
