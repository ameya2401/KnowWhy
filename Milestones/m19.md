# AUDIT LOGS

Track every critical platform action.

Include:

- User Login
- Organization Created
- Project Created
- Integration Connected
- AI Query Executed
- Automation Executed
- Settings Updated
- Permission Changes

Each log entry should include:

- Timestamp
- User
- Organization
- Project
- Action
- Result
- IP Address (if available)

Support filtering and searching.
