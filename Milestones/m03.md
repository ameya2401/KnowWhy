# AUTHENTICATION REQUIREMENTS

Implement:

## User Model

Create a production-ready User model.

Include:

- UUID
- Full Name
- Email
- Profile Picture
- Provider (Google/GitHub)
- Provider ID
- Is Active
- Last Login
- Created At
- Updated At

Email must be unique.

---

## OAuth

Support:

- Google OAuth

- GitHub OAuth

Follow OAuth 2.0 best practices.

Do not implement username/password authentication.

KnowWhy is OAuth-first.

---

## JWT

Implement:

- Access Token
- Refresh Token

Proper signing.

Token verification.
