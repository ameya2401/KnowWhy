# PROJECT CONTEXT

KnowWhy is an Organizational Intelligence Platform.

The Dashboard is the primary workspace users see after selecting a project.

Unlike traditional dashboards, KnowWhy focuses on providing context instead of overwhelming users with charts.

The dashboard should answer:

- What is happening?
- What changed recently?
- What requires attention?
- What knowledge exists?

Design the dashboard to become the central navigation point for future AI features.

---

# OBJECTIVE

Build a modern Workspace Dashboard that provides an overview of project activity, connected integrations, and recent engineering updates.

Focus on usability, clarity, and extensibility.
