# PROJECT CONTEXT

KnowWhy now has:

- Structured Knowledge
- Hybrid Search
- Embeddings
- Organizational Memory

However, KnowWhy cannot yet reason over retrieved information.

The objective of this milestone is to build the Intelligence Engine responsible for constructing high-quality context and generating grounded AI responses.

This milestone implements Retrieval-Augmented Generation (RAG), but in a modular, provider-agnostic architecture.

Do NOT build a chatbot interface.

That belongs to the next milestone.

---

# OBJECTIVE

Implement the KnowWhy Intelligence Engine.

The engine should:

- Understand user questions.
- Retrieve relevant organizational knowledge.
- Build optimized prompts.
- Query an LLM.
- Verify retrieved sources.
- Generate evidence-backed responses.

The Intelligence Engine should remain completely independent of the chat interface.
