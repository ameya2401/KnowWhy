# REQUIREMENTS

Implement:

## Database Configuration

Configure SQLAlchemy using modern SQLAlchemy 2.x patterns.

Use:

- Declarative Base
- Async Engine (preferred)
- Async Session
- Session Dependency

---

## Alembic

Configure Alembic completely.

Requirements:

- Initial migration
- Migration environment
- Versioning support
- Auto-generation enabled

---

## Base Model

Create a reusable BaseModel that every database model will inherit.

Include common fields:

- id (UUID)
- created_at
- updated_at

Use automatic timestamps.

---

## Database Utilities

Implement:

- Database Session
- Connection Management
- Health Check
- Startup Validation

---

## Configuration

Use Pydantic Settings.

Configuration must support:

- Development
- Testing
- Production

No hardcoded values.

---

## Health Endpoint
