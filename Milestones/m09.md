# PROJECT CONTEXT

KnowWhy builds Organizational Memory by collecting knowledge from multiple platforms.

Google Drive represents the organization's document repository.

This integration is responsible for importing and synchronizing documents so they can later become part of KnowWhy' Knowledge Store.

Do NOT generate embeddings.

Do NOT perform AI processing.

Only collect, normalize, and store metadata and supported document content.
